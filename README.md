## 📌 Descripción

**GouPets Store** es una API REST desarrollada con Spring Boot que permite la gestión completa de inventario de productos para una tienda de mascotas.

El sistema implementa operaciones CRUD (Create, Read, Update, Delete) para administrar productos, incluyendo nombre, descripción, precio y cantidad disponible.

Este proyecto fue desarrollado aplicando buenas prácticas como arquitectura en capas y separación de responsabilidades.

---

## 🚀 Características

- ✅ Crear productos
- 📋 Listar todos los productos
- 🔍 Buscar producto por ID
- ✏️ Actualizar productos
- 🗑 Eliminar productos
- 📦 Control de stock
- 🏗 Arquitectura en capas

---

## 🛠 Tecnologías Utilizadas

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Maven
- Base de datos relacional (MySQL / H2)
